## 4. Юнит-экономика и МАРЖА

**Полный стоимостный прогон sample01 (~33 мин) на текущем стеке:**

| Стадия | Стоимость | Доля |
|---|---|---|
| 0 Import | $0 | — |
| 1 Transcribe (Deepgram) | ~$0.14 | ~90% |
| 2 Select (Gemini flash) | ~$0.016 | ~10% |
| 3 Reframe / 4 Captions / 5 Render | $0 (local CPU) | — |
| **ИТОГО** | **~$0.16 / видео (33 мин)** | |

- Экстраполяция на **60 мин**: Deepgram ~$0.26 + Gemini ~$0.03 = **~$0.29/видео**.
- **Таргеты плана:** cost/video ≤ **$2** (ceiling $3). Сейчас **~$0.16–0.29** — глубоко в пределах.
- **Маржа:** при цене $5–15/видео и cost ~$0.16–0.29 → **gross margin ~94–99%** (без инфры).
- **Доминанта стоимости = транскрипция** ($0.14 из $0.16). Главный рычаг маржи —
  self-host WhisperX (→ $0) и кэш транскрипции по hash(source) при итерациях промпта.
- Инфра (не в cost/video): Railway (worker) + Vercel (web) — фикс/мес, добавить при деплое.

## 5. Качество: как меряем (рубрика §7.1)

Поклипово 0/1 по C1–C8 (standalone, hook, clean start/end, face-framing, captions
accuracy/sync, длина), бакеты USABLE/FIXABLE/REJECT, метрика **Q = (usable+fixable)/total**.
GO-gate: Q ≥ 60%, usable ≥ 25%, reason_agrees ≥ 70%. Полную оценку (golden set 6 источников,
≥18 клипов) делаем на этапе H/J, результаты — отдельной секцией сюда.
